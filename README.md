# PPS - Personal Performance System

PPS is a gamified habit tracker and personal performance system by UpaLakshya Labs.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Supabase

## Local Development

```sh
npm install
npm run dev
```

The development server runs on:

```txt
http://localhost:8080
```

## Quality Checks

```sh
npm run lint
npm run test
npm run build
```

## Deployment

Build the production version with:

```sh
npm run build
```

Deploy the generated `dist` folder to your preferred hosting provider.
